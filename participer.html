Contribuez au développement du client Minecraft en rejoignant notre communauté.

Comment Participer

Nous accueillons toutes les contributions, qu'il s'agisse de développement, de tests, de documentation ou de promotion. Voici comment vous pouvez aider :

Développement

Si vous êtes un développeur Java avec une expérience en modding Minecraft, vous pouvez contribuer en écrivant du code. Nous utilisons GitHub pour gérer les contributions.

Voir le Dépôt GitHub

Tests

Aidez-nous à tester les nouvelles fonctionnalités et à identifier les bugs. Les testeurs jouent un rôle crucial pour assurer la stabilité du client.

Signaler un Bug

Documentation

Contribuez à la documentation en rédigeant des guides d'utilisation, des tutoriels ou en améliorant les informations existantes. Une bonne documentation est essentielle pour faciliter l'adoption du projet.

Contribuer à la Documentation

Promotion

Aidez à faire connaître le projet en le partageant sur les réseaux sociaux, en écrivant des articles de blog, ou en réalisant des vidéos de présentation.

Partager sur Twitter
